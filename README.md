# 6 Sınıflı Görüntü Tanıma Projesi

Bu proje bir görüntüyü şu sınıflardan birine ayırır:
- Cadde
- Deniz
- Orman
- Dağ
- İç Mekan
- Tarihi Yer

## Yöntem
Görüntünün renk dağılımı, parlaklığı ve kontrastı kullanılarak
basit bir sınıflandırma yapılmıştır.

## Kullanılan Teknolojiler
- Python
- Gradio
- NumPy
- Pillow

## Çalıştırma
```bash
pip install -r requirements.txt
python app.py
